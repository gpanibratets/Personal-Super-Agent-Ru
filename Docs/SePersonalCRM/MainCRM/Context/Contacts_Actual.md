# Contacts Actual - Активные контакты партнеров и клиентов

**Назначение файла**: Быстрая справка по всем активным контактам в пайплайне работы с партнерами и клиентами. Формат: компактные ключевые слова/теги, разделенные запятыми, для мгновенной ассоциации. Используется как агентами, так и людьми для:

- Быстрого восстановления контекста ("ах, это тот человек, который...")
- Поиска релевантных людей по темам/связям
- Составления сводок задач для последующих действий
- Подготовки контекста перед звонком/встречей

**Формат записи**:

- **Имя** (роль/компания) - ключевые слова, разделенные запятыми, дата последнего контакта
- **Задачи от контакта** → что они обещали/должны сделать
- **Задачи с нашей стороны** → что нам нужно сделать
- **Статус** → текущая стадия взаимодействия
